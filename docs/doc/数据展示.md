## CList

#### 参数说明

|  参数   | 类型  | 默认值  | 说明  |
| :----: | ----  | ----  | :----  |
| legend  | String | - | 备选值有 primary, danger, warn, plain-primary, plain-default, plain-warn, plain-danger, disabled |
| title  | String | - | mini/default |
| icon  | String | - | 同原生button |
| href  | String | - | 布尔值，true/false |
